getSpectrumLabelsByLabels(measureLabel, durationLabel, firstSpaceLabel)

Get all the labels as a hierarchical cascade. For examples, if you feel the 3 arguments for a dataset that have a second-space dimension of size greater than 1, it will return an array [ "the measure label", "the duration label", "the first space label", ["1st label of 2nd sp" "2nd label of 2nd sp", "3rd label of 2nd sp"]]. Only the last argument (the array) is not already given in argument but this sort of repeat is so that the returned values has all the cascade.

Parameters

  • String measureLabel :

    the desired label within the 'measure' dimension (optional)

  • String durationLabel :

    the desired label within the 'duration' dimension (optional)

  • String firstSpaceLabel :

    the desired label within the 'first-space' dimension (optional)

QeegModFileParser

An instance of QeegModFileParser can be used to parse several file (you don't need to create a QeegModFileParser instance per file to parse). The Qeeg MOD file usually have the .MOD extension, though this parser does not need the filename or the extension.
